Detailed Description

Manages local caching of unit data and assets for the Analogiq plugin.

This singleton class handles caching of unit JSON definitions, faceplate images, thumbnails, and control assets to improve performance and enable offline usage. The cache is stored in the user's application data directory and mirrors the remote structure for consistency.
